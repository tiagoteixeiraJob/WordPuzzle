# Motivation

```
This program is created to solve a word puzzle following this steps:

    - Include a StartWord.
    - Include a EndWord.
    - Come up list of words that move from the StartWord to the EndWord.
    - Only one letter can change between any two words.
    - Each intermediate step should be a word from the dictionary file.
```

# Example

```
If StartWord = Spin, EndWord = Spot and DictionaryFile file contains:
Spin
Spit
Spat
Spot
Span

Then ResultFile should contain:
Spin
Spit
Spot
```
